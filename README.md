# ModernWeb
Opensource Repository Holds all the DEMO projects that can help the .NET Beginner to embark and understand well the .NET ecosystem application development especially Backend with deep understanding of how all components exist and communicate with each other to for any kind of application.

# What are Involved
- Minimal API design
- ASP.NET Core
- Balzor
- Razor Page with ASP.NET Core APP
- Http Client
- Unity Testing
